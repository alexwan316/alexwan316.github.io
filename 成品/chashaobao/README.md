1.叉燒餡材料：

叉燒 200克
乾蔥 2粒
洋蔥 1/2個


2.調味料：
生抽 1茶匙
蠔油 1湯匙
糖 2 茶匙
粟粉（玉米粉） 2茶匙
水 4湯匙
麻油 少許


3.塗麵包面材料：
蛋黃 1個（打勻）
糖水（1茶匙糖+2茶匙水）


4.叉燒餡做法:

把叉燒，乾蔥，洋蔥分別切粒備用。

燒熱油鑊，炒香洋蔥至軟身，下乾蔥爆香，跟著倒進叉燒粒兜炒一會，就倒進調味料煮至濃稠即成。盛起備用。

跟這個食譜做麵包，發好，分成6等分，把叉燒餡料包入。放入已預熱至200度C的焗爐中，焗約15分鐘後，拿出焗爐掃上蛋黃汁，再放入焗爐焗多5分鐘，然後塗上糖水即成。
